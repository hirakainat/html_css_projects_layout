# Contact Form

A simple contact form web page with name, email, message, and subject fields, along with company location details.

## Features

- Contact form with Name, Email, Message, and Subject fields
- Contact info section showing address, phone number, and email
- Map image display
- Clean, two-column layout

## Tech Stack

- HTML5
- CSS3

## Project Structure

```
contact_form/
│
├── index.html      # Main HTML structure
└── style.css         # Styling
```

## How to Run

1. Clone or download this repository
2. Open `index.html` in any web browser
