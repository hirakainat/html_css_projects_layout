# Greatzone - Blog Website

A simple, responsive blog landing page built with HTML and CSS. The project features a clean hero section and a grid-based blog card layout to showcase articles.

## Features

- **Navigation Bar** – Brand logo with navigation links and social media icons (Instagram, LinkedIn, Twitter)
- **Hero Section** – Eye-catching heading with supporting text and an image
- **Blog Cards Grid** – A 3-column responsive grid displaying blog post previews, each with an image, title, excerpt, date, and "read more" link
- **Hover Effects** – Smooth opacity transition on the "read more" link

## Tech Stack

- HTML5
- CSS3 (Nested CSS syntax)

## Project Structure

```
blog_website/
│
├── index.html      # Main HTML structure
└── style.css        # Styling and layout
```

## How to Run

1. Clone or download this repository
2. Open `index.html` in any web browser

No build tools or dependencies required — pure HTML & CSS.

## Notes

- Image paths in `index.html` currently point to a local Downloads folder (`C:Users/Hira/Downloads/...`). Update these paths to a relative `images/` folder before deploying or sharing the project, so images load correctly on any system.
