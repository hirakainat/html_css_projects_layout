# Percentage Calculator

A simple web app that calculates your total marks and percentage based on 4 subjects.

## Features

- Enter marks for Maths, English, Islamyat, and Science
- Click "Calculate" to instantly see your total marks and percentage
- Clean, card-style UI with a background image

## Tech Stack

- HTML5
- CSS3
- JavaScript

## Project Structure

```
calculator/
│
├── index.html      # Main HTML structure
├── style.css         # Styling
├── script.js         # Calculation logic
└── background.png    # Background image
```

## How to Run

1. Clone or download this repository
2. Open `index.html` in any web browser
3. Enter your marks and click "Calculate"
